How Reminders Work Admin
  • Each benefit has its own reminder triggers (e.g. 7/3/1 days before period end), set per user from the benefit's page.
  • An email goes to each user's notification address when today matches a trigger and they still have unused credit on that benefit.
  • Users opt out of a benefit via its Ignore toggle — there's no global off-switch.
  • Each trigger fires at most once per user per period.
